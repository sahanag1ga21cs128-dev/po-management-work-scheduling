# PO Management Work Scheduling System

A role-based system for managing purchase orders, staff records, project associates, and work scheduling.

## Features

* Admin Dashboard
* Staff Management
* Project Associate Work Tracking
* Work Progress Updates
* Delay Reason Tracking
* Document Upload System
* Soft Delete & Restore
* Dashboard Analytics

## Tech Stack

Frontend: React + Vite + Tailwind CSS
Backend: Node.js + Express
Database: Supabase

## Modules

* Admin Dashboard
* Staff Records
* Project Associates
* Work Scheduling
* Work Documents
* Trash Management
